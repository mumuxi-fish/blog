p5.js 创作技巧：从草图到作品

p5.js 是我用得最多的创意编程工具之一。分享一些在实践中积累的小技巧，希望能对同样在用 p5.js 创作的朋友有帮助。

1. 色彩调色板

不要直接在代码里写颜色值。先定义一个调色板：

const palette = [
  [231, 111, 81],   // 珊瑚
  [244, 162, 97],   // 杏色
  [233, 196, 106],  // 金黄
  [42, 157, 143],   // 青绿
  [38, 70, 83]      // 深蓝
];

function pick() {
  return random(palette);
}

这样换配色只需改 palette 数组，不需要翻遍整个代码。而且用随机选取 + 有限调色盘的方式，既能保持和谐，又有变化。

2. 噪声运动

用 random() 做的运动往往跳动不自然。用 Perlin 噪声会平滑很多：

let t = 0;

function draw() {
  let x = noise(t) * width;
  let y = noise(t + 1000) * height;
  // 使用两个不同的偏移量得到独立的 x/y 噪声值
  t += 0.01;
}

调整增量（0.01 这个值）可以控制运动速度。增量越小，运动越慢越平滑。

3. 画布自适应

在不同屏幕上都保持好效果：

function setup() {
  createCanvas(windowWidth, windowHeight);
}

function windowResized() {
  resizeCanvas(windowWidth, windowHeight);
}

结合 width 和 height 的相对单位来布局元素，可以让你的作品在各种屏幕上都表现良好。

4. 保存输出

看到满意的一帧，别犹豫：

function keyPressed() {
  if (key === 's') {
    saveCanvas('my-artwork', 'png');
  }
}

加这个只需两行代码，但能帮你记录下很多偶然的美妙瞬间。

p5.js 的魅力在于它的即时反馈。改一行代码，画面就变了。多实验，多尝试，最好的作品往往来自意料之外的组合。
